# YourMIUI

## 介绍 
一款优化 `MIUI` 系统中各种不友好功能的模块，基于 [LSPosed](https://github.com/LSPosed/LSPosed)

## 功能列表
- [x] 阻止结束特定软件进程
- [x] 移除多个系统软件的开屏广告和应用内广告
- [x] 禁用应用安装器的安全检测
- [x] 取消设置中二次确认对话框的倒计时
- [ ] 移除设置中需要插入Sim才能进行下一步操作的限制
- [x] 禁用手机管家中不重要的扫描项
- [x] 修复网络助手的校正总是失败的问题
- [x] 清除设置中恶心的系统新版本标记
- [x] 禁止超级壁纸暂停
- [ ] 修复主题壁纸软件的开屏画面抢占屏幕和存在开屏广告的问题
